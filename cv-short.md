# Roentgen for Docs (MedGemma Diagnostic Bot)

A secure, two-part system for medical image analysis using the MedGemma-1.5-4B model. It features a scalable, remote-agnostic FastAPI inference dispatcher and a cloud-hosted Telegram bot interface, connected via a secure reverse SSH tunnel. Designed with strict whitelisting, session isolation, and advanced interactive Region of Interest (ROI) image preprocessing to ensure privacy and high diagnostic accuracy.
